# LaTeX Math Support

LaTeX math is supported using MathJax.

Inline math looks like $f(x) = x^2$. The input for this is `$f(x) = x^2$`. Use `$…$`.

For a block of math, use `$$…$$` on separate lines

```
$$
F(x) = \int^a_b \frac{1}{2}x^4
$$
```

gives

$$
F(x) = \int^a_b \frac{1}{2}x^4
$$
